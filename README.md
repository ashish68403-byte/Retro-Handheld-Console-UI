# Retro Handheld Console UI 🎮

A retro-style handheld gaming console UI built using pure HTML and CSS.

## 🚀 Features
- Retro handheld console design
- Glowing screen effect
- Animated "PRESS START" text
- D-pad controller buttons
- A and B action buttons

## 🛠️ Tech Stack
- HTML5
- CSS3
- Animate.css

## 📂 Project Structure
Retro-Handheld-Console-UI/
 ├── index.html
 └── style.css

## 🎯 Purpose
This project was created to practice CSS layout, flexbox, grid, and UI design.
