## 📖 Description

**Retro Handheld Console UI** is a frontend project that faithfully recreates the look and feel of a classic handheld gaming device using only HTML and CSS. It features a glowing screen effect, a blinking "PRESS START" animation, a functional-looking D-pad, and A/B action buttons — all styled to evoke the golden era of portable gaming. No frameworks, no JavaScript, just clean and expressive CSS.

## ✨ Features

- 🕹️ **Classic D-pad** — Directional pad with up, down, left, and right buttons
- 🔴 **A/B Action Buttons** — Styled action buttons with a retro aesthetic
- 💡 **Glowing Screen Effect** — CSS box-shadow and glow animations simulate a lit LCD screen
- ✍️ **Animated "PRESS START" Text** — Blinking text animation for authentic retro vibes
- 📐 **Fully Responsive Layout** — Centered and adaptable across various screen sizes
- 🎨 **Pure HTML & CSS** — Zero JavaScript, zero frameworks, zero dependencies

## 🎯 Purpose of the Project

This project was built to:

- **Strengthen CSS skills** — including advanced use of flexbox, CSS variables, keyframe animations, and box-shadow techniques
- **Demonstrate UI creativity** — pushing the boundaries of what's possible with pure HTML and CSS
- **Build a portfolio piece** — showcasing frontend design ability without relying on JavaScript or external libraries
- **Pay homage to retro gaming culture** — bringing back the aesthetic of classic handheld consoles in a modern browser context

## 💻 How to Run Locally

No installation or build process required. Just follow these steps:

**1. Clone the repository**

```bash
git clone https://github.com/ashish68403-byte/Retro-Handheld-Console-UI.git
```

**2. Navigate into the project folder**

```bash
cd Retro-Handheld-Console-UI
```

**3. Open in your browser**

```bash
# Option A — Open directly
open index.html

# Option B — Use VS Code Live Server
# Right-click index.html → "Open with Live Server"
```

That's it — no `npm install`, no dependencies, no build step. ✅
